# Іба Чотко — Офіційний мем-сайт

Цей репозиторій містить простий статичний сайт, присвячений мем-персонажу **Іба Чотко**.

Особливості:
- Мемний візуальний стиль (яскраві кольори, неонові відблиски).
- Логотип у SVG (assets/logo.svg).
- Інтерактивна мем-зона — плитки з фразами, які вимовляються через Web Speech API.
- Кнопка для швидкого програвання фрази та генератор випадкових фраз.
- Зазначено творця сайту: **Головнін В.С.**

Як запустити:
1. Скопіюйте файли на свій вебсервер або просто відкрийте `index.html` у браузері.
2. Для найкращого голосового синтезу використовуйте сучасний браузер (Chrome, Edge, Firefox).
3. Якщо Web Speech API не доступний у браузері — при натисканні буде показано сповіщення.

Автор контенту сайту:
- Творець цього сайту — Головнін В.С.

Ліцензія та використання:
- Матеріали сайту — приклад, створений для демонстрації. Якщо ви плануєте використовувати фрази або графіку у комерційних цілях — домовтесь із автором.
